# LPZero

Personal site of Laxmikant Patil — live at **[lpatil.dev](https://lpatil.dev)**.

A terminal-styled single-user blog. Hand-written HTML and CSS, no build step,
deployed via GitHub Pages.

## Structure

```
├── index.html           # home (terminal prompt, photo, bio, socials)
├── about/               # bio + career timeline
├── travel/              # travel notes (coming soon)
├── photography/         # photo gallery (coming soon)
├── tech/                # technical writing (coming soon)
├── assets/
│   └── style.css        # shared stylesheet — single source of truth
├── pictures/            # portrait and post images
└── CNAME                # custom domain mapping
```

## Run locally

```
open index.html
```

That's it. No server, no dependencies.

## Stack

- HTML, CSS — no framework
- [JetBrains Mono](https://www.jetbrains.com/lp/mono/) via Google Fonts
- Inline SVG for icons and brand marks
- GitHub Pages for hosting
